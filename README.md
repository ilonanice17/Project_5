# \## Описание программы:

# Графическое приложение "Weather Diary" на Python, разработанное с использованием Tkinter. Позволяет вести дневник погоды, записывая дату, температуру, описание и информацию об осадках. Поддерживает добавление, просмотр, удаление записей, фильтрацию по дате и температуре, а также сохранение и загрузку данных в формате JSON.

# 

# \## Функционал:

# \*   Добавление новых записей о погоде.

# \*   Просмотр всех записей в табличном виде.

# \*   Фильтрация записей по дате.

# \*   Фильтрация записей по температуре (выше заданного значения).

# \*   Удаление выбранной записи.

# \*   Очистка всего списка записей.

# \*   Сохранение данных в файл `weather\_diary.json`.

# \*   Автоматическая загрузка данных при запуске приложения.

# \*   Валидация вводимых данных.

# 

# \## Требования:

# \*   Python 3.7+

# \*   Tkinter (обычно входит в стандартную установку Python)

# \*   tkcalendar (для удобного выбора даты) - `pip install tkcalendar`

#

# \## Примеры использования:

# 

# \### Добавление записи:

# 1\.  Выберите дату с помощью календаря.

# 2\.  Введите температуру (например, `15.5` или `-3`).

# 3\.  Введите краткое описание (например, "Солнечно, небольшой ветер").

# 4\.  Выберите "Да" или "Нет" в поле "Осадки".

# 5\.  Нажмите кнопку "Добавить запись".

# 

# \### Фильтрация:

# 1\.  В разделе "Фильтры":

# &#x20;   \*   Выберите конкретную дату с помощью календаря.

# &#x20;   \*   Введите минимальную температуру (например, `10`), чтобы увидеть записи с температурой выше +10°C.

# 2\.  Нажмите кнопку "Применить фильтр".

# 3\.  Для сброса фильтров нажмите "Сбросить".

# 

# \### Удаление и очистка:

# \*   Для удаления одной записи: выберите ее в таблице и нажмите "Удалить выбранную".

# \*   Для удаления всех записей: нажмите "Очистить все" и подтвердите действие.

# 

# \### Сохранение:

# \*   Данные автоматически сохраняются в файл `weather\_diary.json` при закрытии приложения.

# \*   Вы также можете принудительно сохранить данные, нажав кнопку "Сохранить".

# 

# \## Тестирование (примеры):

# 

# \### Позитивные случаи:

# \*   Добавление записи с текущей датой, температурой +20.5, описанием "Ясно", осадками "Нет".

# \*   Очистка всех записей после подтверждения.

# \*   Попытка добавить запись с пустой температурой.

# \*   Попытка добавить запись с пустым описанием.
